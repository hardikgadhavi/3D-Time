# 3D Clock Animation

A beautiful animated 3D clock built using HTML, CSS, and JavaScript. It demonstrates how to use transform and animation properties in CSS to create a 3D rotating clock effect.

## Features
- Real-time clock
- 3D rotation using CSS
- Smooth animations

## Technologies Used
- HTML
- CSS
- JavaScript
